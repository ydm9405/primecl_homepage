처음 배우는 생성형AI 프로그램개발(LLM)

LLM 애플리케이션이 사람들의 관심을 끌 수밖에 없는 이유는 무엇일까요? 현재까지 수많은 기술들이 등장했다 사라져 갔는데, 오늘날에도 여전히 사용되고 있는 주요 기술은 개발이나 사용 패러다임을 크게 발전시켰다는 특징이 있습니다. LLM 애플리케이션이 주력으로 자리 잡아 개발자 두뇌 속의 필수 공구상자에 들어갈지를 판단하기 위해, 우선 어떤 변화가 일어나고 있는지를 전문 강사님의 강의를 통해 알아봅시다!

● 학습 목표

  1. 1. LLM 아키텍처에 대해 기초를 다질 수 있다.
  2. 2. RAG라는 LLM의 대표적인 애플리케이션을 만드는 방법에 대해 파악할 수 있다.
  3. 3. 멀티 모달과 에이전트 고급 주제 다루는 기술까지 확인할 수 있다.

● 교육 특징

  1. 1. 트랜스포머 아키텍처부터 RAG 개발, 모델 학습, 배포, 최적화, 운영까지! 누구나 활용할 수 있는 AI 애플리케이션 개발의 모든 것!

    LLM 시대를 맞이하여 필수적으로 갖춰야 하는 개발 지식을 이론과 실무 양쪽 관점에서 모두 확인할 수 있도록 내용 구성

  2. 2. 전문 강사의 LLM 기술의 노하우를 다양한 자료와 실습 영상으로 구성

    전문 강사의 LLM을 활용하는 방법과 노하우를 다양한 자료를 통해 학습자들이 학습에 몰입할 수 있도록 하고, 직접 실습하는 영상을 보여주며, 학습자들이 이해하기 쉽도록 강의 구성

  3. 3. ‘LLM을 활용한 실전 AI 애플리케이션 개발’ 도서 제공을 통한 학습 내용 복기

    강사님의 ‘LLM을 활용한 실전 AI 애플리케이션 개발’ 도서를 제공하여 복습의 기회 제공

● 교육 대상

  1. 1. 기업체 전 임직원
  2. 2. LLM을 배우고 싶은 직장인
  3. 3. LLM을 활용해 AI 애플리케이션을 개발하려는 개발자

● 강의 목차

  1. 제1장 LLM 지도
  2. 제2장 트랜스포머 아키텍처와 말 잘듣는 모델 학습
  3. 제3장 효율적인 GPU 활용과 sLLM 학습
  4. 제4장 모델 가볍게 만드는 방법과 sLLM 서빙
  5. 제5장 LLM 애플리케이션 개발과 임베딩 모델 활용
  6. 제6장 RAG 개선과 구현
  7. 제7장 멀티 모달 생성 실습
  8. 제8장 멀티 모달 LLM과 에이전트
